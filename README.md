# WoW Fury Warrior PS (Midnight 12.x)

Fury Warrior rotation plugin for **World of Warcraft (Midnight 12.x)**, supporting both talent trees:
- **Slayer**
- **Mountain Thane**

## Features
- Rotation logic tailored for Fury Warrior
- Support for both **Slayer** and **Mountain Thane** builds
- Designed for use as a **PS plugin**

## Requirements
- World of Warcraft **12.x (Midnight)**
- PS (and PS plugin support enabled)

## Installation

### Option A — Install from GitHub
1. Download this repository (Code → Download ZIP, or clone it).
2. Extract/copy the addon/plugin folder into your PS plugins directory:
   - Put the plugin inside your PS **`plugins`** folder (the folder PS uses to load plugins).
3. Launch PS (or reload plugins) and enable the plugin.

### Option B — Install from the PS site
1. Download the plugin directly from the PS site.
2. Enable it in PS.

## Usage
1. Ensure you are on a **Fury Warrior**.
2. Select your talent build (**Slayer** or **Mountain Thane**).
3. Enable the rotation in PS and adjust any PS settings you prefer (keybinds, toggles, etc.).

## Notes
- This repository is written in **Lua**.
- If you run into issues, please include your:
